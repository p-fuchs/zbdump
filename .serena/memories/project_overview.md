Python 3.12 project named zbdump. Current repo is very small: pyproject.toml, zbdump.py, README.md, and Docker/Compose fixture files for PostgreSQL 17 manual testing. The Python app uses Click for CLI handling and psycopg2 for PostgreSQL access. A PostgreSQL 17 fixture database can be started with Docker Compose and contains one seeded table named public.dump_fixture plus multiple index shapes.
